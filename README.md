##说明

1. 广告的模块的class命名，我们统一用GG（广告）的拼音开头，千万别用 ad 这两个字母（会被屏蔽）。比如：http://www.18touch.com/app/ 这里面的图片`<img alt="超强插件" src="images/ad1.png" class="nkjhyldryjrbmhwsyjvv">` 会被adblock屏蔽。
1. “孤零零”的一个图片，最好加上alt和title，不然也非常容易被屏蔽。
